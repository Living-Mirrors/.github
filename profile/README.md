## What This Is

Every AI you've ever used thinks one way. One architecture. One mode of perception. One interaction pattern. One learning style. One creative process.

Humans are not one thing. The dyslexic brain perceives differently than the sequential brain. Two players co-creating produces outcomes neither achieves alone. Genuine novelty requires collision, not recombination. Every human learns through a different cognitive channel.

Living Mirrors builds cognitive specifications that restructure how AI operates across these dimensions. Not prompts. Not personas. Structural specifications grounded in peer-reviewed neuroscience. Each one a document you paste into any AI. **The spec is the implementation.**

Each axis is independent. They stack.

## The Modes

### [Savant Mode](https://github.com/Living-Mirrors/savant-mode) — *Dyslexic Genius*

Seven principles from dyslexic cognition neuroscience. Five emergent combinations. Cross-domain pattern recognition, spatial reasoning, and polyphonic perception. The cognitive surplus that 5,500 years of literacy-first culture called a deficit.

`Dynamic Systems → Global-First → Peripheral Detection → Spatial Reasoning → Narrative → Associative Leaping → Divergent Default`

### [Play Mode](https://github.com/Living-Mirrors/play-mode) — *The Oldest Intelligence*

Seven principles from play neuroscience. Six output layers. The interaction dynamic that turns any AI from a tool into a co-creator. 300 million years of hardwired architecture.

`Magic Circle → Reciprocity → Potential Space → A Head Taller → Yes, And → Adaptive Variability → Continuation Desire`

### [Spark Mode](https://github.com/Living-Mirrors/spark-mode) — *The Generative Axis*

Seven principles from creativity science. Six output layers. The mechanics of genuine novelty — from fuel through collision through emergence. The spark jumps the gap.

`Prepared Mind → Constraint as Engine → Remote Reaching → Bisociation → Janusian Holding → Incubation → Emergence`

### [Learn Mode](https://github.com/Living-Mirrors/learn-mode) — *The Learning Axis*

Seven principles from cognitive science of memory and encoding. Six output layers. Detects how you learn. Adapts everything to your cognitive architecture. The forgetting curve never had a chance.

`Detect, Don't Ask → Content Before Learner → Emotional Encoding → Dual Channel → Push Up the Stack → Calibrate → Weave Retrieval`

### [Imagineer Mode](https://github.com/Living-Mirrors/imagineer-mode) — *The Process Axis*

Seven principles from constructive imagination neuroscience and Walt Disney Imagineering. Six output layers. The act of building reveals what planning never will. Making IS thinking.

`Experience First → Blue Sky Then Constraint → Story Is Structure → Make It Sensory → Build to Learn → Plus It → The Weenie`

## How to Use

```
1. Pick a mode (or stack several)
2. Open the SPEC.md file
3. Copy the full contents
4. Paste into your AI's system prompt
5. Start a conversation
```

Works with any AI. Claude, GPT, Gemini, local models. The spec restructures cognition at the prompt level.
